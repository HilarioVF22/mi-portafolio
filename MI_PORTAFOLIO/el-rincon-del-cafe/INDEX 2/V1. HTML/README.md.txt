# El Rincón del Café — Versión 1 (HTML Puro)

Primera versión del proyecto: estructura HTML básica, sin estilos ni scripts.

## Contenido
- Estructura semántica (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`)
- Tabla con `<thead>`, `<tbody>` y `<tfoot>`
- Formulario funcional con validación mínima
- Etiquetas: `<address>`, `<time>`, `<ruby>`, `<code>`, `<u>`, `<s>`
- Comentarios de organización
